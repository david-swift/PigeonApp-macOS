**EXTENSION**

# `URL`
```swift
extension URL
```

## Methods
### `string(_:)`

```swift
public static func string(_ url: String) -> Self
```

Get the URL from a string. Use this function **only** when you are sure that the string is a valid URL.
- Parameter url: The URL as a string.
- Returns: The URL.

#### Parameters

| Name | Description |
| ---- | ----------- |
| url | The URL as a string. |
