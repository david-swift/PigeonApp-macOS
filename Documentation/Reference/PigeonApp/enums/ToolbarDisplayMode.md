**ENUM**

# `ToolbarDisplayMode`

The display mode of the elements in the toolbar.

## Cases
### `iconOnly`

Show only icons.

### `iconAndText`

Show icons and labels.

### `textOnly`

Show only labels.

## Properties
### `id`

The identifier.

### `localized`

A localized description of the display mode.

### `asNSDisplayMode`

Convert ``ToolbarDisplayMode`` to AppKit's NSToolbar.DisplayMode.
